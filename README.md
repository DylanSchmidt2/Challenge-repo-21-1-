## Installation
To run the app locally, the necessary dependencies can be installed by running..
```
npm i
```
### (Dependencies)
* Boostrap
* Apollo Server
* Express
* GraphQL
## Usage
To run the app locally run..
```
npm run develop
```
or
```
npm run start
```
